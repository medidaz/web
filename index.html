Olá! Este é o Medidaz.

O aplicativo que permite que você consulte medidas caseiras de alimentos brasileiros de A a Z.

  • Teste aqui!

Aplicativo online

O Medidaz pode ser instalado e utilizado em diversos sistemas operacionais. Mas, para utilizar o aplicativo é preciso um serviço de tráfego de dados ativo.

Banco de dados

O Medidaz disponibiliza medidas caseiras de diferentes referências bibliográficas brasileiras. Para identificá-las, utilizamos o padrão ABNT.

Novidades sempre

O Medidaz é um aplicativo dinâmico e estará sempre passando por revisões, validações e recebendo cadastros novos.
